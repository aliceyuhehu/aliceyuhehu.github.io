Breakout Ball Game


The breakout ball game is Java singleplayer console game based on the classic 1970s Atari Inc. arcade game, where the player must use a digital paddle to launch a moving ball and destroy a layer of bricks that line the top third of the screen.

The VSEPR Notation Calculator is a program made with HTML and JavaScript that determines the VSEPR (Valence Shell Electron Pair Repulsion) notation and molecular geometry of a simple or polyatomic molecular compound when its chemical formula is inputted.

Snake


The Snake Game is a Java single-player console game in which the player controls a snake that moves around a bordered plane while trying to avoid hitting the edges of the playing area. Each time the snake eats a piece of food, its tail grows longer, making the game increasingly difficult.

Skills: Java, Java GUI, AWT, JFrame, JPanel, Swing, OOP
